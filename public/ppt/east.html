Slides-Dynamique
PLAN I - Présentation du projet II - Technologies utilisées III - Outil collaboratif GIT IV – Fonctionnalités implémentées V – Difficultés rencontrées VI – Évolutions futures Conclusion Démo
I - Présentation du projet Objectifs : Permettre la diffusion de slides d’un poste maître vers des postes esclaves dans un réseau privé (Intranet par exemple) ou sur internet Avec la Synchronisation de vidéos L’ enregistrement et la rediffusion de sessions
I - Présentation du projet EAST est un outil de présentation WYSIWYG Utilisant uniquement les technologies du Web Multiplateforme (entièrement portable)
II - Technologies utilisées Utilisation de technologies web libres : Nodes.js (utilisation des web sockets, de nombreux modules) JavaScript JQuery HTML5/CSS3 (templates, Session storage, …)
III - Outil collaboratif GIT Développement autour de GIT : Création d’un dépôt GIT Création d’issues Soumission de pullsrequests NotreGitHub : https://github.com/OresteVisari/Slides-Dynamiques WIKI : https://github.com/OresteVisari/Slides-Dynamiques/wiki
IV – Fonctionnalités implémentées Mise en place d’un système d’authentification Sélection de nouvelles présentations Accès rapide à toutes les présentations déjà sélectionnées Synchronisation de vidéo Enregistrement et Diffusion de session EAST enregistrée Réalisation d’un tchat privé Gestion des droits (root, animateur, utilisateur) Mise en place d’une couche SSL
V – Difficultés rencontrées Prise en main du code Montée en compétence sur les nouvelles technologies liées à Node.js De nombreux bugs à corriger
VI – Evolutions futures Responsive design (tablette, smartphone, etc) Intégration de la voix Mise en place d’une application télécommande pour gérer les slides à distance Possibilité de naviguer lorsque l’on rejoue une session Possibilité d’uploader un .odt(intégration du projet EAST-convertor) Intégration de la voix synchronisée avec la session
VI – Conclusion Bon projet en équipe Découverte du travail collaboratif Bonne montée en compétence sur les web sockets et les technologies liées à Node.js
